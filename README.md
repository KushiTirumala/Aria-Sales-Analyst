Sales Analyst AI Agent

An AI-powered Sales Analyst Agent that analyzes sales documents, files, or Excel sheets and provides insights based on user queries. Supports multiple file inputs for efficient, automated sales analysis.

Project Overview

This project demonstrates how AI and Python can be combined to automate sales data analysis. The agent reads multiple files, processes structured and unstructured data, and answers user questions, helping teams make faster, data-driven decisions.

Features

Analyze Excel, CSV, and text-based sales documents

Handle multiple files at once

Query-based insights: ask questions and get answers from the data

Automated data extraction and summarization

User-friendly interface

Technologies Used

Python 3.x

Pandas & NumPy (data handling)

OpenAI / NLP models (AI query processing)

File handling for multiple inputs

Data summarization & analysis logic
